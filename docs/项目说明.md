# 北邮云邮教学空间 · 作业待办（项目说明）

完整目录树见 **[STRUCTURE.md](./STRUCTURE.md)**。

## 建议存放位置

例如：`C:\Users\XuZhunzhi\bupt-uclass-homework-widget`（即 **`%USERPROFILE%\bupt-uclass-homework-widget`**）。不要在桌面长期留副本，以免路径混乱。

## 用途

- 登录 [云邮教学空间](https://ucloud.bupt.edu.cn/uclass/) 后，用 **Python + Playwright** 抓取学生端待办（站点为需登录 SPA）。
- **Electron**：默认打开**客户端主页**；**桌面小组件**为独立置顶小窗，在主页中打开。主页可设置**开机自启动**、**启动后自动同步一次**（偏好写入本机 `electron_prefs.json`）。另可选 `bupt-hw.cmd widget`（tkinter）。

## 环境与首次运行

1. `python -m venv .venv`，激活后 `pip install -r requirements.txt`，`python -m playwright install chromium`，`npm install`。
2. 若无 `config.yaml`：复制 `python/config.example.yaml` 到**仓库根目录**并改名为 `config.yaml`（与 `homework_cache.json`、`browser_profile` 同级）。
3. 推荐在客户端主页点「登录教学空间」完成认证；若仍可用 **`首次登录.bat`** 走 Playwright Chromium。
4. **`启动应用.bat`** 或 `npm start`（先打开主页）；在主页或桌面小组件内点「立即同步」。命令行也可用 **`bupt-hw.cmd fetch`**。

## 数据放哪里

- **开发/源码运行**：默认 **仓库根目录**（`paths.py`：脚本在 `python/` 下时，数据目录为上一级）。
- **打包后的 exe**：Electron 设置环境变量 **`BUPT_DATA_DIR`** 为 `%APPDATA%\北邮作业待办`，配置与缓存、调试文件写到这里；**Playwright 用户目录 `browser_profile` 仍在该目录下**（相对路径由 `config.yaml` 决定）。

打包后的 exe **不包含** Python 解释器与 `.venv`。使用 portable 目录时，请将本机已配置好的 **`.venv` 复制到与 `北邮作业待办.exe` 同一层**，或安装系统 Python 并自行 `pip install playwright PyYAML` 且能访问 `resources/python` 下的脚本（不推荐手动改）；**更简单的是继续用源码目录 + `npm start`**。

## 打包 Windows exe

在仓库根目录：

```powershell
npm install
npm run dist
```

产物在 **`release/`**：`北邮作业待办 Setup x.x.x.exe`（安装版）与 **`北邮作业待办 x.x.x.exe`（portable 绿色版）**。首次使用 portable 前仍需在本机准备好 Python 依赖（见上），或将开发环境的 `.venv` 拷到 exe 旁。
